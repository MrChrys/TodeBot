## Estrutura

O Tode foi feito usando a linguagem **JavaScript** e usa a livraria **Discors.js** para se conectar com o discord >:3, o banco de dados eu utilizo o **Quick DB**.

#### **Arquivo** `.env`
```
TOKEN=

WEBHOOK_ID=
WEBHOOK_TOKEN=
```

#### Dependências
`discord.js@12.5.1`

`dotenv@8.2.0`

`moment@2.29.1`

`parse-ms@2.1.0`

`quick.db@7.1.3`
